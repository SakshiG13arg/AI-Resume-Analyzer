# AI Resume Analyzer 🚀

An intelligent web application that analyzes resumes using NLP and provides skill insights, scoring, and improvement suggestions.

## 🔥 Features

- 📄 Upload Resume (PDF)
- 🧠 NLP-based Text Processing (spaCy)
- 🎯 Role-based Skill Detection
- 📊 Resume Scoring System
- 🔍 Keyword Detection
- 💡 Personalized Suggestions

## 🛠️ Tech Stack

- Python
- Streamlit
- spaCy
- pdfplumber

## 🚀 How It Works

1. Upload your resume
2. Select your desired job role
3. The system:
   - Extracts text from PDF
   - Cleans and processes text using NLP
   - Detects relevant skills
   - Calculates a resume score
   - Identifies important keywords
   - Provides suggestions for improvement

## 📊 Output

- Skills detected
- Resume score (with feedback)
- Keywords found
- Suggestions to improve resume
